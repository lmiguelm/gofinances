## 💻 Como usar

- **_Clone o projeto:_** <br>
  ` git clone https://github.com/lmiguelm/gofinances`

- **_Entre na pasta do projeto:_** <br>
  `cd gofinances`

- **_Baixe as dependências:_** <br>
  ` yarn || npm install`

- **_Rode o app:_** <br>
  `expo start`
